# Sistema de Provas Online

Sistema profissional de provas desenvolvido em Python + Streamlit.

## Funcionalidades

- Cadastro de alunos
- Login com senha
- Recuperação de senha
- Ranking público
- Histórico individual
- Dashboard de evolução
- Cronômetro de prova
- Download de erros

## Tecnologias

- Python
- Streamlit
- Pandas
- Excel / Google Sheets
